# JM-section1-task7

## Features
- Babel
- SCSS
- Font loading
- Autoprefixer
- Sourcemaps

## Requirements
- [Node](https://nodejs.org/)

## Usage
##### Clone this repository
```
git clone https://github.com/alex-karo/webpack-static-template PROJECT-NAME
cd PROJECT-NAME
```
##### Install npm 
```
npm install
```
##### Start

To start the development server
```
npm start
```
To build for production
```
npm run build
```
